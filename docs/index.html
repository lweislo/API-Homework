What is the weather like as you approach the equator?

It gets hotter and I can prove it - just look at my plots!

In this project, I created a Python script to visualize the weather of 500+ cities across the world of varying distance from the equator.

Using Python's Numpy, Pandas, and CityPy modules and the OpenWeatherMap API, I was able to generate a dataset that was representative of global land masses.

I then charted the data using Seaborn and Matplotlib, comparing values in regions where measurements were taken in daytime and night, looking at the following comparisons:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude

I was able to demonstrate that there is an inverse relationship between absolute value of latitude and temperature.
